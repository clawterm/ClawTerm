ClawtermA terminal for managing AI coding agents

Vertical tabs with live status, split panes, per-pane context tracking, desktop notifications, and auto-updates.

Get started
Configuration reference
GitHub

Many agents at once

Vertical tabs show idle / running / waiting / errored state for every Claude Code (or similar) session, so you can fan out and glance at progress without clicking through each one.

Split panes

Up to eight panes per tab — split horizontally or vertically to compare runs, watch a server log next to a coding session, or pair an agent with a shell.

Git worktrees

One agent per branch, isolated by default in a sibling-of-repo worktree namespace. The legacy in-repo layout is supported for power users who want it.

Configurable shortcuts

Every shortcut is editable in place. Disable bindings you don't want, search the full list, and rebind on the fly.

Auto-updates

Checks for new releases hourly by default. Manual check from settings; auto-install opt-in.

Open source

MIT licensed. Built on Tauri v2, xterm.js, and a vendored tauri-pty.
